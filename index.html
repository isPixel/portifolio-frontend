Projetos

Web scraping

Desenvolvi um projeto de web scraping focado em sites de imóveis, coletando automaticamente informações como preço, localização e disponibilidade. O sistema organiza esses dados em relatórios detalhados, facilitando análises imobiliárias e tornando a pesquisa de ofertas mais eficiente.

Módulo de rastreio - App

Desenvolvi um módulo de rastreamento em React Native que permite monitorar a interação do usuário com a aplicação de forma intuitiva e eficiente. Este módulo registra cada local por onde o usuário passa e as ações que ele realiza, enviando essas informações diretamente para o Firebase Analytics.

Api NestJs

APIs que desenvolvi utilizando o framework NestJS. Com uma arquitetura modular e escalável, o NestJS permite a criação de aplicações robustas e de alto desempenho, facilitando a integração com diferentes serviços e bancos de dados.
